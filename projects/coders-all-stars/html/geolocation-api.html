# Le GeoLocation API :

L'API de GeoLocalisation du HTML5, permet d'obtenir des informations sur la localisation de l'Utilisateur depuis son Navigateur.

Toutefois, celui-ci doit approuver au préalable cette demande de fonctionnalité.

Trouver les informations de Localisation :
navigator.geolocation.getCurrentPosition();

NOTES ::

Les Paramètres,,

Les Retours par le Navigateur, peuvent se faire de 2 manières...

  1. GeoDetic : Obtention des Coordonnées Géographiques, sous forme numérique principalement.
  2. Civic : Réception des informations, sous la forme d'adresse, avec le nom du Pays etc.

La Fonction getCurrentPosition(); renvoit toujours un Object. Celui-ci contient les Coordonnées avec Latitude, Longitude et autres infos de ce type.
