NOTES ::

Tous les Navigateurs ne prennent pas en charge les mêmes formats de fichiers Audio et Vidéo.

En incluant plusieurs formats de fichiers, le Navigateur passe d'un élément listé à un autre si le fichier n'est pas compatible.

Les attributs (sont les mêmes, une nouvelle fois) :

On peut se servir dans ce cas, des éléments suivants...
  • controls : affiche les boutons PLAY | PAUSE | STOP (etc.)
  • autoplay : le fichier démarre dès que la page est chargée.
  • loop : la vidéo est lancée en boucle.

Les principaux formats de fichiers Vidéo sont actuellement : MP4 | WebM et OGG.
